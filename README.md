# Aditya Diwan — Personal Developer Portfolio

A dark editorial, bold typographic personal portfolio website built with React, Tailwind CSS, Framer Motion, and Lenis.

## About the Project
This project is a premium personal portfolio highlighting my design craft, interface interactions, and engineering work. It contains components for:
* **Interactive Hero**: Clean luxury editorial style headings with typing role rotation.
* **Work Timeline & Details**: Interactive list of services and professional history.
* **Projects Grid**: Bento-styled layout of selected works.
* **Live Dashboard Graphs**: Mocked contribution activity graph.
* **Command Palette**: Accessibility menu (`Ctrl + K`) for quick actions.

## Tech Stack & Core Libraries

* **React 19**: Frontend UI rendering framework.
* **Tailwind CSS & Tailwind Animate**: Utility-first styling framework and animations.
* **CRACO**: Custom Webpack alias and build configurations.
* **Lenis**: Momentum-based Awwwards-style smooth scrolling.
* **Framer Motion**: Smooth entrance transitions and cursor tracking.
* **React Parallax Tilt**: 3D parallax hover card glare effects.
* **React Fast Marquee**: Infinite horizontal text scrolling banners.
* **Sonner**: Modern pop-up toast notifications.
* **Lucide React**: Clean vector line icons.
