# Memory Companion - React Frontend

This is the React frontend for the Memory Companion app, styled using Tailwind CSS and structured with a glassmorphic dark-theme UI.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm start
   ```

3. Build the application for production:
   ```bash
   npm run build
   ```
